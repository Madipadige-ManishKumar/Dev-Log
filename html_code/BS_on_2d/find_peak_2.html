Problem Statement
Find a 2D Peak element. An element is a peak if: mat[i][j] > its neighbors (Up, Down, Left, Right).
Binary Search Logic
1. Binary search on columns.
2. Find the row index of the maximum element in the mid_col.
3. Since it's the max in its column, it's already > its Top/Bottom neighbors.
4. Compare it to Left/Right neighbors and move the search range.
Python Implementation
