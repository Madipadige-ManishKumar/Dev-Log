Problem Statement

Given a boolean 2D array where each row is sorted, find the 0-based index of the first row that has the maximum number of 1s.

Optimal Logic: O(N + M)
def rowWithMax1s(mat):
    n, m = len(mat), len(mat[0])
    max_row = -1
    j = m - 1 # Start top-right
    
    for i in range(n):
        while j >= 0 and mat[i][j] == 1:
            j -= 1 # Move left
            max_row = i
            
    return max_row
Start at Top-Right corner. Move Left on 1, Move Down on 0.
