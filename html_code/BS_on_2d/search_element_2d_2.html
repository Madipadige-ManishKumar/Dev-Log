Problem Statement
Given an M x N matrix where integers in each row and column are sorted in ascending order, search for a target value.
Optimal Strategy (O(M + N))
We use the Staircase Search. By starting at the Top-Right:
  • If current value > Target: All elements below it are even larger. Move Left.
  • If current value < Target: All elements to its left are even smaller. Move Down.
Python Implementation
def searchMatrix(matrix, target):
    # Start at Top-Right corner
    row, col = 0, len(matrix[0]) - 1
    
    while row < len(matrix) and col >= 0:
        if matrix[row][col] == target:
            return True
        elif matrix[row][col] > target:
            col -= 1  # Move Left
        else:
            row += 1  # Move Down
            
    return False
