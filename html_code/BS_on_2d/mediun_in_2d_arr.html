Problem Statement
Find the median of an M x N matrix (odd size). Each row is sorted.
The median is the element that has (M*N)//2 elements smaller or equal to it.
The Logic
1. Range is [min(col 0), max(last col)].
2. Pick mid_val from range.
3. Use Upper Bound on each row to count elements ≤ mid_val.
4. If total_count ≤ half, median is higher.
Python Implementation
while low <= high:
    mid = (low + high) // 2
    count = 0
    for row in matrix:
        # Binary search within each row
        count += bisect_right(row, mid)
    
    if count <= (M * N) // 2:
        low = mid + 1
    else:
        high = mid - 1
return low
Initial Range: [1, 40]. Median position: 4 (Total 9 elements).
