Problem Statement

Write an efficient algorithm that searches for a target value in an m x n integer matrix. Rows are sorted left-to-right, and the first integer of each row is greater than the last integer of the previous row.

Python Implementation

def searchMatrix(matrix, target):
    m, n = len(matrix), len(matrix[0])
    low, high = 0, (m * n) - 1
    
    while low <= high:
        mid = (low + high) // 2
        # Map 1D index to 2D row/col
        r, c = mid // n, mid % n
        
        if matrix[r][c] == target:
            return True
        elif matrix[r][c] < target:
            low = mid + 1
        else:
            high = mid - 1
    return False
