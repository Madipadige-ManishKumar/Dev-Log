Kth Missing Number

PS: Given sorted array arr and integer k, find the kth missing positive number.

Logic: At index i, the number of missing elements is arr[i] - (i + 1).

def findKthPositive(arr, k):
  low, high = 0, len(arr) - 1
  while low <= high:
    mid = (low + high) // 2
    missing = arr[mid] - (mid + 1)
    if missing < k:
      low = mid + 1
    else:
      high = mid - 1
  return low + k
