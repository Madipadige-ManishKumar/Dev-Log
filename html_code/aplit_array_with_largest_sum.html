Split Array Largest Sum

Task: Split array nums into k contiguous subarrays. Minimize the maximum sum among these subarrays.
The Split Logic: We "walk" through the array. If adding the next number makes the current sum > mid, we must split.
def splitArray(nums, k):
  low, high = max(nums), sum(nums)
  while low <= high:
    mid = (low + high) // 2
    chunks, current_sum = 1, 0
    for n in nums:
      if current_sum + n <= mid:
        current_sum += n
      else:
        chunks += 1
        current_sum = n
    if chunks <= k: # Valid
      ans = mid
      high = mid - 1
    else: low = mid + 1
