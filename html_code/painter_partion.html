Painter's Partition

Task: Divide N boards among K painters. Minimize the maximum time any painter spends. Each painter paints contiguous boards.
Threshold Check: If a painter's cumulative board length exceeds mid, we stop them and assign the current board to the next painter.
def paintBoards(boards, k):
  low, high = max(boards), sum(boards)
  while low <= high:
    mid = (low + high) // 2
    if isPossible(boards, k, mid):
      ans = mid
      high = mid - 1
    else:
      low = mid + 1
  return ans
