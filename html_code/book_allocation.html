Book Allocation

Problem Statement: Allocate N books to M students contiguously. We must minimize the maximum pages any student carries.
The Bucket Rule: Think of the mid value as a Bucket Size. We pour books into a student's bucket until the next book would cause an overflow. Only then do we move to the next student.
def countStudents(books, limit):
  students, current_sum = 1, 0
  for b in books:
    if current_sum + b <= limit:
      current_sum += b
    else: # Split Happens
      students += 1
      current_sum = b
  return students
