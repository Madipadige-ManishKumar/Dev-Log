Gas Station Placement

Task: Minimize the maximum distance between adjacent gas stations by adding K new stations.
The Math: If a gap between two stations is dist, and we want to ensure no sub-gap exceeds mid, we need floor(dist / mid) new stations.
def minMaxDistance(stations, k):
  low, high = 0, stations[-1] - stations[0]
  for _ in range(100): # Precision loop
    mid = (low + high) / 2
    if canPlace(mid, stations, k):
      high = mid
    else:
      low = mid
  return high
