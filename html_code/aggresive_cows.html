Aggressive Cows

Problem Statement

Place K cows in stalls at positions stalls[i] such that the minimum distance between any two is as large as possible.

  • Sorted: Stalls must be sorted to use greedy checking.
  • Binary Search Range: 1 to (Max position - Min position).
  • Goal: Maximize the minimum.
def aggressiveCows(stalls, k):
  stalls.sort()
  low, high = 1, stalls[-1] - stalls[0]
  while low <= high:
    mid = (low + high) // 2
# canPlace(mid) greedy logic:
    if canPlace(mid, k):
      ans = mid
      low = mid + 1
    else:
      high = mid - 1
  return ans
